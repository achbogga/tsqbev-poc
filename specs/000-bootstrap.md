# Spec 000: Bootstrap

## Goal

Create a minimal, public, evidence-grounded repo before any autonomous experimentation.

## Requirements

- `docs/plan.md` exists first.
- Every substantive subsystem is traceable to an original paper or official codebase.
- The auto-research loop is disabled.
- The bootstrap yields a functional local repo, not just notes.

## Acceptance

- package metadata exists
- install path is defined through `uv`
- reference matrix exists
- tests can be added and run locally
