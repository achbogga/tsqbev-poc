# RTX 5000 Benchmarks

These measurements were collected on March 26, 2026 on the local development workstation:

- GPU: `Quadro RTX 5000`
- VRAM: `16384 MiB`
- Driver: `570.211.01`
- PyTorch: `2.7.1+cu126`
- TensorRT: `10.16.0.72`

## Measurement Scope

- Batch size: `1`
- Views: `6`
- Image shape: `256 x 704`
- Full-model benchmark:
  - includes the current PyTorch LiDAR seed encoder and camera path
  - uses the default `ModelConfig`
  - uses synthetic tensors already resident on GPU before timing
- TensorRT benchmark:
  - covers the current exportable deployment core only
  - consumes prepared LiDAR query seeds and prepared camera proposal seeds
  - does **not** include raw LiDAR pillarization or proposal backprojection in the TensorRT graph

## Caveats

- These are synthetic fixed-shape latency measurements, not dataset evaluation results.
- The TensorRT path benchmarks the current deployable core only, not the entire end-to-end multimodal pipeline.
- These numbers should not be reinterpreted as Orin numbers. They are workstation measurements for the local `Quadro RTX 5000`.
